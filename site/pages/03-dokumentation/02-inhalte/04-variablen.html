Variablen

Herbie durchläuft deine ganze Website und arbeitet Textdateien mit einem YAML Front-Matter Block ab. Für jede dieser Datei erzeugt Herbie verschiedene Daten und macht diese über das Twig Templating System verfügbar. Nachfolgend siehst du eine Übersicht dieser Daten.
Variable Beschreibung
route Die angefragte Seite, z.B. `doku/vars`.
site Projektweite Variablen.
page Variablen, die der aktuellen Seite zugeordnet werden.
paginator Ein Paginatorobjekt zum Erzeugen einer Pagination.

Site-Variablen

Variable Beschreibung
site.time Die aktuelle Zeit im ISO 8601 Format (2004-02-12T15:19:21+00:00).
site.lastCreated Die Zeit der zuletzt erstellten Seite im ISO 8601 Format (2004-02-12T15:19:21+00:00).
site.lastModified Die Zeit der zuletzt geänderten Seite im ISO 8601 Format (2004-02-12T15:19:21+00:00).
site.menu Ein Menu als Liste aller vorhandenen Seiten.
site.tree Ein Menu als Baumstruktur aller vorhandenen Seiten.
site.rootPath Ein Menu als Liste aller Seiten von der Wurzel bis zur aktuellen Seite.

Page-Variablen

Variable Beschreibung
page.title Der Titel der Seite.
page.layout Das Layout der Seite.
page.type Der Typ der Seite (definiert den Formatierer, z.B. Markdown oder Textile).
page.date Das Datum der Seite.
page.[DATEN] Jede weitere Seitenvariable, welche im Front-Matter Block definiert wurde, ist nutzbar (z.B. page.css, page.keywords, etc.)
